# Task 1: Product Scoping

This folder contains the product-scoping work for the internal marketing performance briefing tool.

## Contents
- `product_brief.md`: main problem statement and proposed solution
- `scope.md`: what is in and out of scope for version 1
- `flow.md`: user journey and interaction flow
- `wireframe.md`: rough screen layout
- `walkthrough.md`: explanation of the reasoning and trade-offs

## Summary
The tool is intended to help internal team members answer the same marketing performance question faster and more consistently. Version 1 focuses on a repeatable summary, trust, and clarity rather than advanced automation.
